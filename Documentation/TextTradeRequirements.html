Introduction

Our team plans to keep the application simple and easy to use. There will be two essential GUIs: one for users, and one for moderators. Each user has to register an account in order to start trading on Text Trade. The user then can create a list of books he wants to buy and/or sells. Each textbook entry must include all the necessary information such as title, author, edition, ISBN, associated course(s) that the textbook is used for, and book condition. A user can search for a book in the marketplace, find potential buyers, communicate with other traders, and report misdeeds to moderator. To keep things neat, we plan to implement the search algorithm based on ISBN numbers and course numbers.
